9. Eliminación de caché de Favicon

Para actualizar el favicon de un sitio web, podemos obligar a los navegadores a descargar una nueva versión agregándola "?v=2" al nombre del archivo.
Esto sería el ejemplo del código:
"link rel="icon" href="/favicon.ico?v=2"/"

18. Etiqueta 'pre'

Se pueda usar esta etiqueta para presentar texto preformateado exactamente como está escrito en el archivo HTML:

Con etiqueta 'pre'

                            display: flex;
                            align-items: center;
                            justify-content: center;   
                        }

Sin etiqueta 'pre'

.center { display: flex; align-items: center; justify-content: center; }
